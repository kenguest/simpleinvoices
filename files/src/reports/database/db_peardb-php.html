db_peardb.php

Authors: Matthew Palmer mpa.nosp@m.lmer@baileyrobe.nosp@m.rts.com.au

A PEAR DB abstraction layer backend for PHPReports.  Acts a little bit differently to other PHPReports backends; I don’t necessarily think that’s a problem at my end.

Easiest way to use this is to pass a DSN into the report via the server parameter.  The rest of the DB parameters are ignored.
